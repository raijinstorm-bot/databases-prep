Which statement is used to remove a single column from an existing table?

- a) `DROP TABLE TableName`
- b) `ALTER TABLE TableName DROP COLUMN ColumnName`
- c) `DELETE FROM TableName WHERE ColumnName`
- d) `DROP COLUMN ColumnName FROM TableName`

### Explanation

A column is removed with `ALTER TABLE TableName DROP COLUMN ColumnName`. DROP TABLE removes the whole table, DELETE removes rows (not columns), and option (d) is not valid syntax.
