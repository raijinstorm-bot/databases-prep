What does WMS stand for, as used to integrate on-the-fly map layers from different organisations?

- a) Wide Map Standard
- b) Web Mapping Schema
- c) Web Map Service
- d) Workstation Map Server

### Explanation

WMS stands for Web Map Service, which can integrate on-the-fly map layers coming from different servers of different organisations to render up-to-date map images. The other expansions are invented.
