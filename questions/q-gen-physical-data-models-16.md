The presentation notes that advanced Oracle CREATE TABLE options (e.g. PCTFREE, STORAGE, TABLESPACE settings). Which statement about them is correct?

- a) They are part of the ANSI SQL standard and portable everywhere
- b) They are DML statements
- c) They must be included in every CREATE TABLE statement
- d) They are usually not portable and often refer to data allocation settings

### Explanation

Such advanced options are usually not portable and in many cases refer to data allocation settings, so the statements run only on one platform. They are not standard/portable, not DML, and not mandatory.
