Which profile resource limit helps mitigate a brute-force attempt to guess a user's password by repeated logon attempts?

- a) IDLE_TIME
- b) PASSWORD_LIFE_TIME
- c) FAILED_LOGIN_ATTEMPTS
- d) SESSIONS_PER_USER

### Explanation

FAILED_LOGIN_ATTEMPTS sets the number of failed logon attempts after which the account is locked, directly countering password-guessing attacks. IDLE_TIME and SESSIONS_PER_USER limit resources, and PASSWORD_LIFE_TIME governs password expiry.
