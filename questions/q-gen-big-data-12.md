Which statements correctly describe sharding as presented?

- a) Data is distributed across multiple servers, with different data on different servers
- b) Each server has its own storage and can fully manage (create, modify, read) its subset of data
- c) Sharding requires every server to hold a complete copy of the entire data set
- d) Sharding is usually combined with replication to avoid having only one copy of the data
- e) Sharding inherently guarantees referential integrity across the cluster

### Explanation

Sharding splits data across servers (a), each managing its own subset (b), and is typically combined with replication for redundancy (d). It does not require full copies on every node (c), and the sharding approach typically offers no referential integrity (e).
