Which of the following is an example of a system privilege rather than an object privilege?

- a) Executing SQL SELECT on a specific table
- b) Inserting rows into HR.DEPARTMENTS
- c) CREATE TABLE
- d) Updating a particular column of a table

### Explanation

System privileges such as CREATE TABLE or GRANT ANY PRIVILEGE authorise broad operations, whereas object privileges permit operations on specific objects (e.g. SELECT/INSERT/UPDATE on a named table). The other options are all object privileges.
