Auditing in Oracle can be used to track which of the following?

- a) Selected actions of selected users or on selected objects
- b) Logon actions, including unsuccessful attempts
- c) The DBA's activity
- d) The exact CPU clock speed of the server

### Explanation

Auditing can track selected actions on selected users/objects (a), both successful and unsuccessful logon attempts (b), and even the DBA's activity (c). Monitoring the server's CPU clock speed is not an auditing function.
