According to the "Why a database?" comparison, what advantages do databases have over flat files (TXT, XML, CSV)?

- a) Databases avoid storing data on disk, keeping everything in memory
- b) Special structures (indexes) speed up information retrieval
- c) Concurrent updates by many users are supported with transactional processing
- d) A complex permission system secures the data

### Explanation

Databases use indexes for fast retrieval (b), support concurrent transactional updates (c), and provide complex permission systems (d). Option (a) is false — databases store data permanently on disk, which is one reason they are used.
