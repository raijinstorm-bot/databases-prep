Which of the following are defining features of a relation (table)?

- a) Each relation has a unique name in a database
- b) Each column has a unique name within the relation
- c) The order of columns is significant and must be preserved
- d) All values in a single column are of the same type

### Explanation

The presentation defines a relation by: a unique relation name, a unique column name per relation, all values in a column being the same type, and that the order of columns is NOT important. Therefore option c contradicts the definition, while a, b and d are correct.
