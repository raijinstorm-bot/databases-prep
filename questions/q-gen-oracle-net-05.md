Which background process is responsible for writing transaction logs?

- a) DBWn
- b) LGWR
- c) ARCn
- d) LREG

### Explanation

LGWR (Log Writer) writes transaction logs. DBWn saves changes to the database, ARCn archives transaction logs, and LREG registers the instance with a listener.
