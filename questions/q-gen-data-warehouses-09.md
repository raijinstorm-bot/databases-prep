In data warehouse terminology, a "fact" is defined as:

- a) a descriptive attribute used to group records
- b) a data element used to measure business performance
- c) a hierarchical set of attributes such as year/quarter/month
- d) a table that describes proper city names

### Explanation

A fact is a data element used to measure business performance (e.g. records of sales, purchases, deliveries). The grouping attributes (a, c) describe dimensions, and the descriptive lookup (d) describes a dimension table.
