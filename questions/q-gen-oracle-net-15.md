Which naming method uses LDAP-compliant directory services so that local tnsnames.ora files need not be maintained on many workstations?

- a) Local naming
- b) External naming
- c) Directory naming
- d) Easy Connect naming

### Explanation

Directory naming uses LDAP-compliant directory services to resolve service aliases and obtain server, protocol, port and instance name, removing the need for local tnsnames.ora files. External naming uses non-Oracle services such as NIS/DCE/CDS, and local naming relies on tnsnames.ora.
