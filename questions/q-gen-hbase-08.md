The lecture describes an HBase table conceptually as which of the following?

- a) A fixed grid of typed columns
- b) A map of maps, where every map is a key-value pair
- c) A graph of nodes and edges
- d) A single JSON document per table

### Explanation

The lecture describes an HBase table as a map of maps, where every map is a key-value pair; keys are arbitrary strings mapping to a row, and a row is itself a map from columns to values (b). The other descriptions match other data models, not HBase.
