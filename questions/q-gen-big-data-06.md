How does the "Big Data approach" differ from the "traditional approach" as presented?

- a) Big Data carefully limits the number of variables to avoid confusing the model
- b) Big Data tries to get as much data as possible and uses the best-fitting models as black boxes
- c) Big Data always develops a theoretical formula explaining the process
- d) Big Data relies primarily on small expert-designed surveys

### Explanation

The Big Data approach gathers as much data as possible, develops many models, and uses the best ones as black boxes (b). Limiting variables, developing explanatory theory, and expert-designed surveys describe the traditional approach.
