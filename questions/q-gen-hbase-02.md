According to the lecture, which statements describe Apache HBase?

- a) It is the leading database-related project in the Hadoop group
- b) Its goal is hosting very large tables (billions of rows by millions of columns) on clusters of commodity hardware
- c) It provides Bigtable-like capabilities on top of Apache Hadoop and HDFS
- d) It is a relational database optimised for ACID transactions
- e) It is designed to run only on a single high-performance server

### Explanation

HBase is the leading Hadoop database project, designed for very large tables on commodity-hardware clusters, providing Bigtable-like capabilities atop Hadoop/HDFS (a, b, c). It is non-relational with limited transactions (d is false) and is distributed across clusters, not single-server (e is false).
