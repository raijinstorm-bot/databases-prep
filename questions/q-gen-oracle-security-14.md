Which value of the AUDIT_TRAIL parameter causes audit records to be written into the database table SYS.AUD$?

- a) NONE
- b) DB
- c) OS
- d) FALSE

### Explanation

Setting AUDIT_TRAIL to DB (or TRUE) saves the audit trail into the SYS.AUD$ database table. NONE/FALSE disable auditing, and OS writes the trail to operating system files.
