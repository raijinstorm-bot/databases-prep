In Ms SQL Server, which service is named as the coordinator responsible for managing the two stages of a distributed transaction?

- a) SQL Server Agent
- b) Microsoft Distributed Transaction Coordinator
- c) Log-reader agent
- d) Distribution agent

### Explanation

The lecture names the Microsoft Distributed Transaction Coordinator as the coordinating service for the two phases of a distributed transaction (b). SQL Server Agent schedules jobs, while the log-reader and distribution agents belong to transactional replication.
