The key objective of creating an index is to improve performance for which operations?

- a) Searching for records matching a WHERE condition
- b) Matching records in a JOIN clause
- c) Sorting records with ORDER BY
- d) Inserting large batches of new records faster

### Explanation

Indexes improve searching (WHERE), joining (JOIN) and sorting (ORDER BY). They generally slow inserts/updates/deletes because the index trees must be kept up to date, so (d) is false.
