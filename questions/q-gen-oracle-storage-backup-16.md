Which statements about Recovery Manager (RMAN) are correct?

- a) It can only create logical backups by querying table data
- b) It can back up data files, control files, archive logs and spfiles
- c) It provides server-managed backups and the most complete functionality
- d) It can determine which data blocks have changed to create incremental backups

### Explanation

RMAN backs up data files, control files, archive logs and spfiles (b), provides server-managed backups with the fullest functionality (c), and can build incremental backups by tracking changed blocks (d). Querying table data for logical backups describes Data Pump, not RMAN, so (a) is false.
