Which HDFS shell command copies a file from the local file system into HDFS?

- a) copyToLocal
- b) get
- c) copyFromLocal
- d) cat

### Explanation

`copyFromLocal` copies a local file to an HDFS location (c). `copyToLocal` and `get` move data the other way (HDFS to local), and `cat` copies source files to stdout.
