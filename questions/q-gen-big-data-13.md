With a replication factor of 3, what is true of each shard?

- a) Each shard exists on exactly one node
- b) Each shard is present on 3 nodes
- c) Each shard is split into 3 unrelated pieces with no copies
- d) The whole cluster is limited to 3 nodes total

### Explanation

A replication factor of 3 means each shard is present on 3 nodes (b), providing redundancy while the cluster itself may contain hundreds of nodes. The other options misinterpret what the replication factor means.
