Why is there no need to transfer data into a data warehouse in an on-line (real-time) manner?

- a) Because source databases are always offline
- b) Because ETL tools cannot operate in real time
- c) Because the warehouse is used to observe trends and aggregate values, for which the most recent hour is not needed
- d) Because the warehouse stores only the current state, never history

### Explanation
