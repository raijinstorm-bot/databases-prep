Which background process is responsible for registering an instance with a listener?

- a) LGWR
- b) ARCn
- c) LREG
- d) DBWn

### Explanation

The LREG process registers an instance with a listener. LGWR writes transaction logs, ARCn archives them, and DBWn writes changes to the datafiles.
