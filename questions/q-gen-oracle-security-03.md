In Oracle, roles are lists of privileges. Which of the following are valid categories of roles mentioned in the presentation?

- a) System-defined roles such as DBA and CONNECT
- b) Object roles such as SELECT and INSERT
- c) User-defined roles
- d) Profile roles such as DEFAULT

### Explanation

The presentation lists two role categories: system-defined roles (e.g. DBA, CONNECT) and user-defined roles. SELECT/INSERT are object privileges, not roles, and DEFAULT is a profile, not a role.
