Which attributes typically comprise a fact table?

- a) the date of the record (e.g. date of a sale)
- b) large binary fields and long descriptive text columns
- c) a number of measure attributes
- d) dimension attributes that are foreign keys to dimension tables
- e) only a single row describing the whole organisation

### Explanation

A fact table typically holds the record date (a), measure attributes (c), and foreign keys to dimensions (d). Because fact tables are very large, text descriptions and large binary fields are usually excluded (b), and they contain many rows, not one (e).
