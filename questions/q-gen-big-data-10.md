Regarding the scalability of relational DBMS for large-scale web data, which statements are correct?

- a) Scaling up a single server indefinitely fully solves the problem
- b) Scaling up (a more powerful server) is not sufficient to deal with the problem
- c) A shared-disk cluster (e.g. Oracle RAC) is still not sufficiently scalable and is expensive
- d) Relational shared-disk clusters scale easily to thousands of nodes
- e) Web-scale processing was never a performance or cost concern

### Explanation

Scaling up is insufficient (b) and shared-disk clusters such as Oracle RAC remain insufficiently scalable and costly (c). Indefinite scale-up does not solve it (a), shared-disk clusters do not reach thousands of nodes (d), and performance/cost limits were the key driving factor (e).
