Which statements about MongoDB are correct according to the lecture?

- a) It is an example of a document-based system
- b) Data are stored as JSON-based documents
- c) It cannot perform any indexing or aggregation
- d) It can work with both aggregate-based and aggregate-ignorant data models, including a $lookup to link collections
- e) SQL is its primary query language

### Explanation

MongoDB is a document store using JSON-based documents (a, b), supports indexing/queries/aggregation, and can link collections via $lookup, handling both aggregate styles (d). It does support indexing and aggregation (so c is false), and SQL is not supported (so e is false).
