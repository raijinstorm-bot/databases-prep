How can HBase tables be accessed, according to the lecture?

- a) Through Java libraries / Java API (the default way)
- b) Via REST requests over the network
- c) Using a C/C++ client or Scala
- d) Through SQL statements by default, with no extra project needed
- e) Only through a graphical desktop application

### Explanation

By default HBase is accessed via Java libraries/API (a), and alternatives include REST over the network (b) and C/C++ or Scala clients (c). SQL is not the default and requires an additional project (so d is false), and access is not limited to a GUI (e is false).
