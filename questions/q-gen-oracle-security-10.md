Which profile setting points to a PL/SQL function implementing a custom password complexity policy?

- a) PASSWORD_REUSE_MAX
- b) PASSWORD_VERIFY_FUNCTION
- c) PASSWORD_GRACE_TIME
- d) PASSWORD_LOCK_TIME

### Explanation

PASSWORD_VERIFY_FUNCTION names a PL/SQL function that implements custom password policy verification. PASSWORD_REUSE_MAX limits reuse count, PASSWORD_GRACE_TIME is the grace period after expiry, and PASSWORD_LOCK_TIME is how long an account stays locked.
