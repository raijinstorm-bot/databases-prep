Which statement best distinguishes a local file system from HDFS?

- a) HDFS is managed by the node's operating system, while the local file system spans many nodes
- b) The local file system is managed by the node's OS, whereas HDFS is managed by Hadoop and typically spans many nodes but appears as a single file system from each node
- c) Both refer to exactly the same storage, just with different commands
- d) HDFS shell statements by default refer to the local file system

### Explanation

The local file system is managed by the node's OS, while HDFS is managed by Hadoop, spans many nodes, yet appears as one file system from each node (b). HDFS shell commands by default refer to HDFS, not the local file system (d), and the two are not the same storage (c).
