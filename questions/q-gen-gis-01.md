According to the presentation, what is a layer (warstwa) in a geodatabase?

- a) A scanned paper map combined with its coordinate metadata
- b) A collection of objects of the same category
- c) A single object together with its descriptive attributes
- d) A network communication protocol such as WMS

### Explanation

The presentation defines a layer as a collection of objects of the same category, for example a layer of rivers or a layer of buildings. A single object is not a layer, and WMS is a service, not a layer.
