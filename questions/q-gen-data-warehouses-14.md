In an OLAP data cube, each cell represents:

- a) a dimension attribute
- b) a measure value
- c) a single source database
- d) an ETL step

### Explanation

In a data cube each cell represents a measure value, while each edge denotes a dimension attribute. The cube is built on a fact table and some dimension tables, not on source databases or ETL steps.
