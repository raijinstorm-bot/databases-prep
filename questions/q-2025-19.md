Sample NoSQL platforms include:

- a) Apache HBase
- b) ESRI ArcGIS
- c) MS SQL Server
- d) MongoDB

### Explanation

Apache HBase (wide-column) and MongoDB (document) are NoSQL platforms (a, d). ESRI ArcGIS is a GIS (b) and MS SQL Server is a relational DBMS (c).
