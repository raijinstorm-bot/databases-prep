During name resolution when a Net service name is used, which settings are identified to establish the connection?

- a) DBMS server name / network address
- b) DBMS instance name
- c) Protocol-specific settings such as the TCP/IP port
- d) The user's password for the database account

### Explanation

Name resolution determines the server address, instance name, protocol-specific settings (e.g. TCP port), and connection-related settings such as dedicated vs shared server. The user's password is supplied by the client at connect time, not obtained through name resolution.
