Which statements about the data stored in a data warehouse are correct?

- a) It contains corporate historical data, in many cases no longer available in OLTP databases
- b) It contains only the most recent transactional records
- c) It includes high-quality data carefully integrated from many sources
- d) It is intentionally kept low-quality to save processing time
- e) It never includes any redundant structures

### Explanation

A warehouse keeps long-term historical data that may already have been purged from OLTP systems (a) and consists of high-quality, carefully integrated data (c). It is not limited to recent records (b), is not low quality (d), and may contain redundant structures to speed up queries (e).
