Which statements about Apache Cassandra are correct according to the lecture?

- a) It relies on a masterless architecture and is highly scalable and resilient
- b) It supports full SQL JOIN operations
- c) It uses CQL, which can be considered a simplified SQL
- d) It is an example of a column-family (wide-column) database
- e) It requires a designated master node to coordinate writes

### Explanation

Cassandra is a masterless, highly scalable, resilient wide-column database querying with CQL, a simplified SQL (a, c, d). CQL does not support JOIN (so b is false), and there is no master node (so e is false).
