Which command is a valid Easy Connect example as shown in the presentation?

- a) `sqlplus marketing/pass@mySalesServer`
- b) `sqlplus marketing/pass@127.0.0.1:1521/orcl`
- c) `sqlplus marketing/pass@//orcl`
- d) `sqlplus marketing/pass`

### Explanation

The Easy Connect syntax is `username/password@host[:port][/service_name][:server_type][/instance_name]`, matching option (b). Option (a) uses a net service name (the alternative method), and the others omit the required host/connection details.
