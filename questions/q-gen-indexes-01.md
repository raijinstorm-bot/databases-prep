What is the primary purpose of an index in a relational database?

- a) To enforce data types on table columns
- b) To speed up the retrieval of rows from a table or view
- c) To store backups of table data on a separate disk
- d) To translate SQL into the DBMS's internal language

### Explanation

An index is an on-disk structure associated with a table or view used to speed up retrieval of rows. It does not enforce data types, store backups, or translate SQL.
