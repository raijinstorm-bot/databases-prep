In the example with columns Name, Surname, PESEL and NIP, where PESEL is the primary key, what role does NIP play?

- a) A foreign key
- b) A non-key attribute that may be NULL
- c) A candidate key
- d) A composite key

### Explanation

The presentation marks PESEL as the primary key and NIP as a candidate key, because NIP is another set of columns whose values are unique for the entire table but was not chosen as the primary key. It is not a foreign key, and as a key it cannot be left empty.
