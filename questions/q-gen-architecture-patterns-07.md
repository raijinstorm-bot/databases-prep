The lecture lists reasons to use distributed databases. Which of the following are mentioned?

- a) For mission-critical systems, a local database can be added when a stable connection to the remote server cannot be guaranteed
- b) To decrease response time
- c) When different software systems require their own DBMS platforms (heterogeneous distributed databases)
- d) To make all data physically reside on a single central server

### Explanation

The lecture cites adding a local database for mission-critical systems with unreliable connectivity (a), decreasing response time (b), and supporting heterogeneous platforms (c). Centralising all data on one server contradicts the very idea of a distributed database, so d is false.
