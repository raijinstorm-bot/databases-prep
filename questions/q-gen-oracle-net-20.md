Regarding the tnsping utility, which statements are correct?

- a) It is used to test connectivity with the listener process
- b) It can accept both Easy Connect parameters and a service name
- c) It does not verify whether the listener actually handles a requested service name
- d) It verifies that the requested service is available before returning

### Explanation

tnsping tests connectivity to the listener and accepts Easy Connect parameters or a service name. It explicitly does not verify the availability of the requested service, nor whether the listener handles a requested service name, so (d) is wrong.
