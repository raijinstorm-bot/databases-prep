According to the presentation, what is the relationship between layers and a GIS project?

- a) Having the required layers is exactly the same as having a GIS project
- b) A GIS project references individual layers but also adds complex settings such as symbols, labels and scale-dependent display rules
- c) A GIS project contains only raster layers, never vector layers
- d) A GIS project is just a single tnsnames.ora-style configuration file

### Explanation

The presentation stresses that having layers is not the same as having a GIS project; a project references layers and adds complex settings (symbols, labels, which layers appear at which scales, data-dependent line widths, etc.). It is not limited to raster layers nor reducible to a single config file.
