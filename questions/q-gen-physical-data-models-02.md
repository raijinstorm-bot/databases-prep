Which of the following are typically classified as DML (Data Manipulation Language) statements?

- a) INSERT
- b) UPDATE
- c) DELETE
- d) CREATE TABLE

### Explanation

INSERT, UPDATE and DELETE manipulate data and are core DML statements (SELECT is sometimes also included). CREATE TABLE defines structure and belongs to DDL, not DML.
