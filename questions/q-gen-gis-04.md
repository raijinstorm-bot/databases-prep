According to the presentation, a single object in a vector layer is composed of:

- a) A shape (e.g. a collection of points)
- b) A location / geographical coordinates
- c) Descriptive attributes such as pipe material or river name
- d) A bitmap tile linked to the region it represents

### Explanation

Each vector map object combines spatial data (shape and location) with descriptive attributes such as pipe material or river name. A bitmap tile is a property of raster layers, not vector objects.
