A product has not been sold yet. According to the presentation, how should its "date of sale" column be represented?

- a) As 0
- b) As NULL
- c) As an empty string ''
- d) As the current date

### Explanation

The presentation uses exactly this example: the date of sale for products not yet sold should be set to NULL, because no proper date could be used in that context. NULL is explicitly distinguished from 0 and from an empty string.
