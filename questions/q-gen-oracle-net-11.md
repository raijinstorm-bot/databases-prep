Which statement about the Easy Connect naming method is correct?

- a) It supports any of the network protocols Oracle Net allows
- b) It requires configuring the client workstation in advance
- c) TCP/IP is the only supported protocol and no client configuration is needed
- d) It requires a tnsnames.ora alias to be defined first

### Explanation

Easy Connect naming requires only server name, TCP port and instance name, needs no client configuration, and supports TCP/IP only with no advanced options. It does not rely on a tnsnames.ora alias, which is the alternative net-service-name method.
