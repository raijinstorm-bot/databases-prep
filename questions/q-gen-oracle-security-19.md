Fine-grained auditing (FGA), managed by the DBMS_FGA package, can monitor SQL statements affecting which of the following?

- a) Selected tables
- b) Selected rows matching a criterion
- c) Selected columns
- d) Selected statement types from SELECT, INSERT, UPDATE, DELETE

### Explanation

FGA provides a standard way to monitor SQL SELECT or DML statements affecting selected tables, selected rows matching a criterion, selected columns, and selected statement types (SELECT/INSERT/UPDATE/DELETE). All four options are correct, and FGA policies are managed via the DBMS_FGA package.
