What is the purpose of the BULK INSERT statement mentioned in the presentation?

- a) To call a stored method in Oracle
- b) To import a datafile into a database (in MS SQL Server)
- c) To insert and update records in one statement
- d) To remove a column from a table

### Explanation

BULK INSERT, in MS SQL Server, is used to import a datafile into a database. Calling a method is CALL (Oracle), the combined insert/update is MERGE, and removing a column is done with ALTER TABLE DROP COLUMN.
